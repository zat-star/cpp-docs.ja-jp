# CDynamicAccessor::Close
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

すべての列をバインドし、割り当てたメモリを解放し、そのクラスの [IAccessor](https://msdn.microsoft.com/en-us/library/ms719672.aspx) インターフェイス ポインターを解放します。  
  
## 構文  
  
```  
  
void Close( ) throw( );  
  
```  
  
## 必要条件  
 **ヘッダー:** atldbcli.h

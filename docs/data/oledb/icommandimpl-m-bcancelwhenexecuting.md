# <a name="icommandimplmbcancelwhenexecuting"></a>ICommandImpl::m_bCancelWhenExecuting
実行するときに、コマンドを取り消すことができるかどうかを示します。  
  
## <a name="syntax"></a>構文  
  
```  
  
unsigned m_bCancelWhenExecuting:1;  
  
```  
  
## <a name="remarks"></a>コメント  
 既定値は**true** (キャンセルできます)。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** atldb.h

# <a name="cdynamicaccessorgetcolumncount"></a>CDynamicAccessor::GetColumnCount
列の数を取得します。  
  
## <a name="syntax"></a>構文  
  
```  
  
DBORDINAL GetColumnCount( ) const throw( );  
  
```  
  
## <a name="return-value"></a>戻り値  
 列の数を取得します。

 **ヘッダー:** atldbcli.h

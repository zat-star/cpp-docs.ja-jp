# CDynamicParameterAccessor::GetParamCount
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

バッファーに格納されているパラメーターの数を取得します。  
  
## 構文  
  
```  
  
DB_UPARAMS GetParamCount( ) const throw( );  
  
```  
  
## 戻り値  
 パラメーターの数。  
  
## 必要条件  
 **ヘッダー:** atldbcli.h

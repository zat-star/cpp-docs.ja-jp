# <a name="comptrinternalrelease-method"></a>ComPtr::InternalRelease メソッド
この ComPtr に関連付けられたインターフェイスに対して COM 解放操作を実行します。  
  
## <a name="syntax"></a>構文  
  
```  
void InternalRelease();  
```  
  
## <a name="remarks"></a>コメント  
 このメソッドは保護されています。  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL

# <a name="comptrcomptr-destructor"></a>ComPtr::~ComPtr デストラクター
ComPtr のインスタンスを初期化解除します。  
  
## <a name="syntax"></a>構文  
  
```  
WRL_NOTHROW ~ComPtr();  
```  
  
## <a name="requirements"></a>必要条件  
 **ヘッダー:** client.h  
  
 **名前空間:** Microsoft::WRL

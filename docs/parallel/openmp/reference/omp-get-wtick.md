# <a name="ompgetwtick"></a>omp_get_wtick
プロセッサのクロックのティック間の秒数を返します。  
  
## <a name="syntax"></a>構文  
  
```  
double omp_get_wtick( );  
```  
  
## <a name="remarks"></a>コメント  
 詳細については、次を参照してください。 [3.3.2 omp_get_wtick 関数](../../../parallel/openmp/3-3-2-omp-get-wtick-function.md)します。  
  
## <a name="example"></a>例  
 参照してください[omp_get_wtime](../../../parallel/openmp/reference/omp-get-wtime.md)を使用する例については`omp_get_wtick`です。

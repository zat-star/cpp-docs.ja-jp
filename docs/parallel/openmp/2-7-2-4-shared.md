# <a name="2724-shared"></a>2.7.2.4 shared
この句に表示される変数を共有する、*変数一覧*チーム内のすべてのスレッド間でします。 チーム内のすべてのスレッドの同じ記憶域にアクセス**共有**変数。  
  
 構文、**共有**句を次に示します。  
  
```  
shared(variable-list)  
```

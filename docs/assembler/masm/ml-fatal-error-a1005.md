# <a name="ml-fatal-error-a1005"></a>ML の致命的なエラー A1005
**アセンブラーの制限: マクロ パラメーター名のテーブルが完全**  
  
 多くのパラメーター、ローカル、またはマクロのラベルは、マクロの定義されました。 マクロ名テーブルに空き領域がありませんがありました。  
  
 短いを定義または少ない名、または不要なマクロを削除します。

# REAL4
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*初期化子*  の各単精度 \(バイト\) の浮動小数点数を割り当て必要に応じて初期化します。  
  
## 構文  
  
```  
  
name REAL4 initializer [[, initializer]]...  
```

# <a name="real4"></a>REAL4
割り当て、必要に応じて各単精度 (4 バイト) の浮動小数点数を初期化*初期化子*です。  
  
## <a name="syntax"></a>構文  
  
```  
  
name REAL4 initializer [[, initializer]]...  
```

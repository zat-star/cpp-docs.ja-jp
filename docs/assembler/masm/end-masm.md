# <a name="end-masm"></a>END (MASM)
モジュールの終了をマークし、必要に応じて、プログラムのエントリ ポイントを設定*アドレス*です。  
  
## <a name="syntax"></a>構文  
  
```  
  
END [[address]]  
```

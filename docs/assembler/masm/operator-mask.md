# operator MASK
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*recordfieldname* または  *レコード*  のビットが設定されそのほかのビットがすべてクリア ビット マスクを返します。  
  
## 構文  
  
```  
  
MASK {recordfieldname | record}  
```

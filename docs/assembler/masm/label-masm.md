# <a name="label-masm"></a>LABEL (MASM)
現在の場所カウンター値を割り当てることによって新しいラベルを作成し、指定された`type`に*名前*です。  
  
## <a name="syntax"></a>構文  
  
```  
  
      name LABEL type  
name LABEL [[NEAR | FAR | PROC]] PTR [[type]]   
```

# ML Nonfatal Error A2107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**\[ラベルにジャンプするか暗黙的な呼び出しを使用できません。**  
  
 とは別のセグメントに近いラベルにジャンプする暗黙的な呼び出しを実行しようとしました。

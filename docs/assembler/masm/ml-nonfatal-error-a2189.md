# ML Nonfatal Error A2189
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**セグメントの配置で無効な組み合わせ**  
  
 **整列**  または `EVEN` のディレクティブで指定された配置は  **セグメント**  のディレクティブで指定されている現在のセグメントの配置を超えています。

# <a name="real8"></a>REAL8
割り当て、必要に応じて各倍精度 (8 バイト) の浮動小数点数を初期化*初期化子*です。  
  
## <a name="syntax"></a>構文  
  
```  
  
name REAL8 initializer [[, initializer]]...  
```

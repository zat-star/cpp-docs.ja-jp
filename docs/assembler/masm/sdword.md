# <a name="sdword"></a>SDWORD
割り当て、必要に応じて各記憶域の符号付きのダブルワード (4 バイト) を初期化`initializer`です。 型指定子としても使用できる、型が有効な任意の場所。  
  
## <a name="syntax"></a>構文  
  
```  
  
name SDWORD initializer [[, initializer]]...  
```

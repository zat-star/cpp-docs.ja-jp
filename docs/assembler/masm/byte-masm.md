# BYTE (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

各 `initializer` のストレージのバイトを割り当て必要に応じて初期化します。  
  
## 構文  
  
```  
[[name]] BYTE initializer [[, initializer]] ...  
```  
  
## 解説  
 型指定子がどこでも可能型であるため使用できます。

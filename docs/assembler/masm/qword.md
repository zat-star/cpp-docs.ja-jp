# <a name="qword"></a>QWORD
割り当て、ごとに 8 バイトの記憶領域を必要に応じて初期化`initializer`です。 型指定子としても使用できる、型が有効な任意の場所。  
  
## <a name="syntax"></a>構文  
  
```  
[[name]] QWORD initializer [[, initializer]]...  
```

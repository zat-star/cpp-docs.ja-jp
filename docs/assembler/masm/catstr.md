# <a name="catstr"></a>CATSTR
テキスト アイテムを連結します。  
  
## <a name="syntax"></a>構文  
  
```  
  
name CATSTR [[textitem1 [[, textitem2]] ...]]  
```  
  
## <a name="remarks"></a>コメント  
 各テキスト アイテムは、リテラル文字列で始まる定数を指定できます、 `%`、またはマクロの関数によって返される文字列。

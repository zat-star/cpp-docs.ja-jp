# <a name="sbyte-masm"></a>SBYTE (MASM)
割り当て、必要に応じて各記憶域の符号付きバイトを初期化`initializer`です。 型指定子として使用することも、型が有効な任意の場所。  
  
## <a name="syntax"></a>構文  
  
```  
  
name SBYTEinitializer [[, initializer]]...  
```

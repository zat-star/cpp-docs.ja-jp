# ENDM
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

マクロまたは繰り返しブロックを終了します。  
  
## 構文  
  
```  
  
ENDM  
  
```  
  
## 解説  
 [マクロ](../Topic/MACRO.md) [FOR](../../assembler/masm/for-masm.md) [FORC](../Topic/FORC.md) [繰り返し](../../assembler/masm/repeat.md)または  **と**  を参照してください。

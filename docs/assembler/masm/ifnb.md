# IFNB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

*textitem が*  null であるアセンブリができます。  
  
## 構文  
  
```  
  
IFNB textitem  
```  
  
## 解説  
 完全な構文については [IF](../../assembler/masm/if-masm.md) を参照してください。

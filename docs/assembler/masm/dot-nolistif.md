# <a name="nolistif"></a>.NOLISTIF
False (0) に評価される条件が条件付きブロックの一覧を表示しません。  
  
## <a name="syntax"></a>構文  
  
```  
  
.NOLISTIF  
  
```  
  
## <a name="remarks"></a>コメント  
 既定値です。 同じ[です。SFCOND](../../assembler/masm/dot-sfcond.md)です。

# YMMWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Intel AVX \(Advanced Vector Extensions\) 命令での 256 ビット マルチメディア オペランドに対して使用します。  
  
## 構文  
  
```  
YMMWORD  
```  
  
## 解説  
 `YMMWORD` は、AVX 組み込みの `__m256` と同じ型を表します。

# <a name="real10"></a>REAL10
割り当て、必要に応じて各 10 バイトの浮動小数点数を初期化*初期化子*です。  
  
## <a name="syntax"></a>構文  
  
```  
  
name REAL10 initializer [[, initializer]]...  
```

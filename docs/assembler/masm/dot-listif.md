# .LISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

false の条件付きブロックのステートメントのリストを呼び出します。  
  
## 構文  
  
```  
  
.LISTIF  
  
```  
  
## 解説  
 [.LFCOND](../../assembler/masm/dot-lfcond.md) と同じです。

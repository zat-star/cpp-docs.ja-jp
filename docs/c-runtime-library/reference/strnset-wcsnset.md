# strnset、wcsnset
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらの POSIX 関数は使用されなくなりました。  代わりに ISO C\+\+ 準拠の [\_strnset、\_strnset\_l、\_wcsnset、\_wcsnset\_l、\_mbsnset、\_mbsnset\_l](../../c-runtime-library/reference/strnset-strnset-l-wcsnset-wcsnset-l-mbsnset-mbsnset-l.md) またはセキュリティが強化された [\_strnset\_s、\_strnset\_s\_l、\_wcsnset\_s、\_wcsnset\_s\_l、\_mbsnset\_s、\_mbsnset\_s\_l](../../c-runtime-library/reference/strnset-s-strnset-s-l-wcsnset-s-wcsnset-s-l-mbsnset-s-mbsnset-s-l.md) を使用してください。

# <a name="bitor"></a>bitor

&#124; 演算子の代替手段。

## <a name="syntax"></a>構文

```C

#define bitor |

```

## <a name="remarks"></a>コメント

マクロにより &#124; 演算子が生成されます。

## <a name="example"></a>例

```cpp
// iso646_bitor.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 1, b = 2, result;

   result = a | b;
   cout << result << endl;

   result= a bitor b;
   cout << result << endl;
}
```

```Output
3
3
```

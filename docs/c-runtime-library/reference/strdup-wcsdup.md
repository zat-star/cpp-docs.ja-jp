# strdup、wcsdup
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

これらの POSIX 関数は使用されなくなりました。  ISO C\+\+ 標準に [\_strdup、\_wcsdup、\_mbsdup](../../c-runtime-library/reference/strdup-wcsdup-mbsdup.md) を代わりに使用します。

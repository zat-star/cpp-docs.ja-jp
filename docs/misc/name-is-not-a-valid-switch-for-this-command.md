# &lt;name&gt; is not a valid switch for this command.
このエラーが発生するのは一般的に、正しくないスイッチまたは単一文字のスイッチの組み合わせがコマンドに入力された場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントでコマンドの正しい構文を確認して、やり直します。

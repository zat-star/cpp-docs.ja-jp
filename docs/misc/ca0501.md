# CA0501
出力レポートを読み取れません  
  
 このエラーは、次のいずれかの理由で出力レポートに書き込むことができない場合に発生します。  
  
-   ディスクがいっぱいです。  
  
-   ディレクトリに書き込みを行うための権限がありません。  
  
-   ディレクトリが存在していません。

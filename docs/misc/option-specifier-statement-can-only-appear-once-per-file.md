# &#39;Option &lt;specifier&gt;&#39; ステートメントは、1 つのファイルに付き 1 回のみ使用できます
`Option Compare` ステートメント、`Option Explicit` ステートメント、`Option Strict` ステートメントは、同じソース ファイル内でそれぞれ一度だけ使用できます。  
  
 **エラー ID:** BC30225  
  
### このエラーを解決するには  
  
-   重複している `Option` ステートメントを削除します。

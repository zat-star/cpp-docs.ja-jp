# CA0053
規則アセンブリを読み込めません。  
  
 これは、規則アセンブリをリフレクションによって読み込むことができなかったことを示します。  このエラーは、ユーザーが持っているカスタム規則が正しく構築されていないか、またはファイル システム アクセスの拒否などの理由で分析ツールがアセンブリを読み込むことができない場合に発生します。

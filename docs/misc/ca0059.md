# CA0059
コマンド ライン スイッチ エラー。  
  
 このエラーは、必要なコマンド ライン スイッチが見つからない場合に発生します。  
  
 たとえば、\/out スイッチや \/console スイッチが指定されていない場合が、これに該当します。  このエラーは、不適切なタイムアウト値など、無効なスイッチがある場合にも発生します。

# An invalid file was specified for /resetsettings: &lt;filename&gt;.  /resetsettings will be ignored.
通常は、設定ファイルが読み取ることができない、存在しない、または不適切な XML が含まれているときに、このエラーが発生します。  
  
### このエラーを解決するには  
  
1.  ファイル名が適切であることを確認します。  
  
2.  ファイルが正しい形式であることを確認します。  
  
3.  ファイルに正しい形式の XML が含まれていることを確認します。

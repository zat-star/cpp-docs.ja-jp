# アプリケーション定義ファイルが互換性のないエディターで既に開かれています。
**アプリケーション定義ファイルが互換性のないエディターで既に開かれています。  そのエディターを終了してから、プロジェクト プロパティ ページを再度読み込んでください。**  
  
 このエラーは、既に別のエディターで開かれているアプリケーション定義ファイルを開こうとした場合に発生します。  
  
### このエラーを解決するには  
  
1.  もう一方のエディターを閉じます。  
  
2.  プロジェクト デザイナーを終了します。  
  
3.  **\[プロジェクト\]** メニューの **\[プロパティ\]** をクリックして、再度プロジェクト デザイナーを開きます。

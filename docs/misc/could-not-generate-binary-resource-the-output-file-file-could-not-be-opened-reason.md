# バイナリ リソースを生成できませんでした。 出力ファイル &#39;file&#39; を開けませんでした。 &lt;理由&gt;
プロジェクト システムによって、古い .resources ファイルのコピーを削除できませんでした。 プロジェクト システムは、リソースを準備する過程で、.NET リソース マネージャー用に、拡張子が .resx の XML ファイルをバイナリの .resources ファイルに変換します。  
  
 このエラーの詳細については、エラー メッセージのテキストの最後に表示されているオペレーティング システム固有の原因をご覧ください。  
  
 **このエラーを解決するには**  
  
-   [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] を再起動します。  
  
     このエラーが発生すると、ビルド処理が失敗します。

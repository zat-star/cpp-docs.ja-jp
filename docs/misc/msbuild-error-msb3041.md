# MSBuild エラー MSB3041
**"'\<ファイル\>'" にマニフェスト リソースを作成できません。**  
  
 `CreateManifestResourceName` タスクが、指定されたファイルのマニフェスト リソース名を作成できません。  
  
### このエラーを解決するには  
  
-   MSBuild エラーの原因と解決方法の詳細については、「[追加のリソース](../Topic/Additional%20MSBuild%20Resources.md)」を参照してください。

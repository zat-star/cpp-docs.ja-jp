# Option is not currently valid.
通常、このエラーが発生するのは、コマンドに対して現在使用できない引数またはスイッチを指定した場合です。  たとえば、「`>Edit.Find test /sel`」と入力した場合に、ファイル内に選択対象がない場合、\/sel スイッチは使用できません。  
  
### このエラーを解決するには  
  
1.  入力した引数やスイッチが現在使用できるかどうかを確認します。  詳細については、「[引数を使用する Visual Studio コマンド](../Topic/Visual%20Studio%20Commands.md)」を参照してください。

# 型 &#39;&lt;typename&gt;&#39; のフィールド &#39;&lt;fieldname&gt;&#39; を持つ構造体のファイル入出力 (I/O) は無効です。
ファイルへの書き込みを試行している構造体には、サポートされていない型が含まれています。  
  
### このエラーを解決するには  
  
-   構造体を確認して、サポートされていない型が検出された場所を判断し、有効な型に変更します。

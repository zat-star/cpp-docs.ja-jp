# MSBuild エラー MSB3189
**MSB3189: このアプリケーションの更新プログラムの場所はローカル パスです。**  
  
 この警告は、配置マニフェストにのみ適用されます。  
  
### このエラーを解決するには  
  
-   更新プログラムの場所が有効な URL であることを確認します。

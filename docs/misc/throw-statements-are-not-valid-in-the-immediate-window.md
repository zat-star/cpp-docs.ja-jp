# &#39;Throw&#39; ステートメントは、イミディエイト ウィンドウでは使用できません
`Throw` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30725  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Throw` ステートメントを削除します。

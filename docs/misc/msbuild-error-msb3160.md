# MSBuild エラー MSB3160
**MSB3160: ファイル '\<ファイル\>' での XML 検証警告です: '\<エラー\>'**  
  
 この警告は、product.xml または package.xml ファイルが XML スキーマに準拠していない場合に生成されます。  この警告の後半部分には、具体的な警告メッセージが示されます。  
  
### このエラーを解決するには  
  
-   XML ファイルを更新して、検証エラーを解決します。

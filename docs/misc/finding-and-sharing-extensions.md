# 拡張機能の検索と共有
**\[ツール\]** メニューから利用可能な **\[拡張機能と更新プログラム\]** を使用して、Visual Studio 拡張機能のダウンロード、インストール、管理が可能です。 拡張機能は、[Visual Studio ギャラリー](http://go.microsoft.com/fwlink/?LinkID=123847) などの Web サイトや他の開発者から直接取得することもできます。 拡張機能の検索とインストールを行う方法の詳細については、「[Visual Studio 拡張機能の検索と使用](../Topic/Finding%20and%20Using%20Visual%20Studio%20Extensions.md)」を参照してください。  
  
 拡張機能を作成し、展開パッケージにラップして、インターネット上での投稿、ネットワーク共有への投稿、または電子メールでの送信により、他の開発者と拡張機能を共有することもできます。 拡張機能を VSIX 展開パッケージにラップし、Visual Studio ギャラリー Web サイトにアップロードすると、他の開発者は**拡張機能と更新プログラム**を使用して拡張機能をインストールできます。 VSIX 展開の詳細については、「[Visual Studio 拡張機能を配布](../Topic/Shipping%20Visual%20Studio%20Extensions.md)」を参照してください。

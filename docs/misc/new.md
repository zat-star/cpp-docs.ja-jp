# new
`new` キーワードの使い方を簡単に説明し、詳細情報へのリンクを紹介します。  
  
|タイトル|説明|  
|----------|--------|  
|[new 演算子](../c-runtime-library/operator-new-crt.md)|ヒープからメモリ ブロックを割り当てます。|  
|[new 演算子](../Topic/operator%20new%20\(%3Cnew%3E\).md)|個々のオブジェクトにストレージを割り当てるため、new 式で呼び出される関数です。|  
|[new 演算子](../cpp/new-operator-cpp.md)|フリー ストアからオブジェクトまたはオブジェクトの配列にメモリを割り当て、そのオブジェクトへのポインター \(適切に型指定されたゼロ以外のポインター\) を返します。|  
|[new \(new slot in vtable\)](../windows/new-new-slot-in-vtable-cpp-component-extensions.md)|C\+\+\/CLI プロジェクトで、仮想メンバーが vtable の新しいスロットを取得することを示します。  つまり、この関数は基底クラスのメソッドをオーバーライドしません。|  
|[ref new, gcnew](../windows/ref-new-gcnew-cpp-component-extensions.md)|[!INCLUDE[cppwrt_short](../Token/cppwrt_short_md.md)]\(`ref new`\) または C\+\+\/CLI \(`gcnew`\) プロジェクトで、この複合キーワードは参照カウント オブジェクトを作成します。|

# MSBuild エラー MSB3173
**MSB3173: マニフェスト '\<ファイル\>' を書き込めません'  \<エラー\>'**  
  
 このエラーは、ビルド処理によるマニフェスト ファイルの書き込み中に一般的な問題が生じた場合に生成されます。  
  
### このエラーを解決するには  
  
-   MSBuild エラーの原因と解決方法の詳細については、「[追加のリソース](../Topic/Additional%20MSBuild%20Resources.md)」を参照してください。

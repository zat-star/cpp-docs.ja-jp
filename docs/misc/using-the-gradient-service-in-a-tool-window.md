# ツール ウィンドウでのグラデーション サービスの使用
[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] のツール ウィンドウが Windows Presentation Foundation でビルドされるため \(WPF\)、コードのグラデーションのサービスを呼び出す必要はありません。  代わりに次に、ツール ウィンドウのサーフェイスを表すデザイナーで、 `StackPanel` の要素を、 **プロパティ** のペインで、グラデーションを設定するように変更します **背景** のプロパティを選択します。  詳細については、「[色、グラデーション、不透明度の設定](../misc/setting-colors-gradients-and-opacity.md)」を参照してください。

# Property パラメーターには &#39;Value&#39; という名前は指定できません
`Property` ステートメントのパラメーター名は `Value`です。 この名前は、`Set` プロシージャのプロパティに割り当てられる新しい値のためにプロパティ定義内で予約されています。  
  
 **エラー ID:** BC32201  
  
### このエラーを解決するには  
  
-   `Property` ステートメントのパラメーターの名前を変更します。

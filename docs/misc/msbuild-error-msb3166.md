# MSBuild エラー MSB3166
**MSB3166: 項目 '\<パッケージ\>' に対する必要なファイル '{0}' が見つかりませんでした。**  
  
 このエラーは、使用許諾契約書 \(EULA: End\-User License Agreement\) または外部チェック ファイルがない場合に生成されます。  このファイルがディスク上に存在することを確認します。  
  
### このエラーを解決するには  
  
-   Visual Studio SDK をインストールし直します。  
  
     または  
  
-   必要なファイルを適切な場所にコピーします。

# &#39;&lt;rsetstmt&gt;&#39; が宣言されていません
'\<rsetstmt\>' が宣言されていません。 RSet ステートメントは現在サポートされていません。Microsoft.VisualBasic.RSet を使用してください。  
  
 以前のバージョンの [!INCLUDE[vbprvb](../Token/vbprvb_md.md)] に組み込まれていたいくつかの関数は <xref:Microsoft.VisualBasic?displayProperty=fullName> 名前空間に移動されています。 これにより、これらの機能をより幅広くすべてのプログラミング言語で使用できます。  
  
 **エラー ID:** BC30821  
  
### このエラーを解決するには  
  
-   `Microsoft.VisualBasic` 名前空間にある `RSet` 関数を代わりに使用します。

# MSBuild エラー MSB2011
**要素 \<{0}\> は有効ではありません。**  
  
 指定された要素が無効です。  
  
### このエラーを解決するには  
  
-   指定した要素のスペルを確認します。  
  
-   プロジェクト ファイルに変更や破損がないことを確認します。  変更または破損がある場合は、プロジェクトを作成するのに使用したバージョンの [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] でプロジェクトをいったん開いて保存し直してから、再び変換してください。

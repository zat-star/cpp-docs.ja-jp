# 配列の次元が、&#39;VBFixedArray&#39; 属性で指定された次元と一致しません
配列の次元が、`VBFixedArray` 属性で指定された次元と一致しません。  
  
### このエラーを解決するには  
  
-   配列の `Dim` ステートメントの配列次元が `VBFixedArray` 属性で指定された次元と一致するようにします。

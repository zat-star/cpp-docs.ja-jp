# モジュールのメンバーは &#39;&lt;specifier&gt;&#39; として宣言できません
`Module` ステートメント内のメンバーに対して正しくない指定子を使用しました。 モジュールは、インスタンス化することができず、継承をサポートしません。また、インターフェイスを実装することもできません。  
  
 **エラー ID:** BC30436  
  
### このエラーを解決するには  
  
-   指定子を削除します。

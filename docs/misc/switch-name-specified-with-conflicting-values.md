# Switch &lt;name&gt; specified with conflicting values.
このエラーは、\[コマンド\] ウィンドウでスイッチを 2 度入力し、各インスタンスでスイッチに指定した値が異なる場合に発生します。  
  
### このエラーを解決するには  
  
-   各スイッチを一度だけ指定して、コマンドを再度入力します。

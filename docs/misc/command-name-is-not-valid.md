# Command &lt;name&gt; is not valid.
このエラーが発生するのは一般的に、Visual Studio に認識されないコマンドまたはコマンドのエイリアスが **\[検索\]** ボックスまたは **\[コマンド\]** ウィンドウに入力された場合です。  エイリアスを入力した場合は、エラー メッセージにはそのエイリアスではなく完全なコマンド名が表示されます。  
  
### このエラーを解決するには  
  
1.  コマンド名を入力し直すか、有効なコマンド名を選択します。

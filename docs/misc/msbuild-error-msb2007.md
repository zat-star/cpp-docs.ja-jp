# MSBuild エラー MSB2007
**Visual Studio では、プロジェクト ファイル "{0}" が見つかりませんでした。**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] で、指定したプロジェクト ファイルが見つかりませんでした。  
  
### このエラーを解決するには  
  
1.  プロジェクト ファイル名のスペルを確認します。

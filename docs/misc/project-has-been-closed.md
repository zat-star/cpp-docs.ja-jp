# プロジェクトは閉じられました。
[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] コンパイラ内でコード モデルのアクセス エラーが発生しました。  
  
 **エラー ID:** BC32301  
  
### このエラーを解決するには  
  
1.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
2.  エラーがまだ発生する場合は、[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] コンパイラを再インストールします。  
  
3.  再インストール後にエラーが続く場合は、状況に関する情報を収集し、Microsoft 製品サポート サービスに通知してください。

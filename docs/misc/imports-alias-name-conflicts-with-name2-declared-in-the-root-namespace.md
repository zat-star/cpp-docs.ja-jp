# Imports エイリアス &#39;&lt;name&gt;&#39; は、ルート名前空間で宣言された &#39;&lt;name2&gt;&#39; と競合しています
`Imports` ステートメントで使用されるエイリアスは、ルート名前空間で宣言されている別の名前と競合しています。  
  
 **エラー ID:** BC31403  
  
### このエラーを解決するには  
  
-   `Imports` ステートメントで使用されるエイリアスに一意の名前を選択します。

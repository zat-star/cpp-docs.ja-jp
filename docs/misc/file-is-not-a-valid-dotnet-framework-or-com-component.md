# File is not a valid .NET Framework or COM component
有効なオブジェクトが含まれていないファイルをオブジェクト ブラウザーで表示しようとしています。  オブジェクト ブラウザーで表示できるのは、[!INCLUDE[dnprdnshort](../Token/dnprdnshort_md.md)] コンポーネントおよび COM コンポーネントだけです。  
  
### このエラーを解決するには  
  
1.  誤ったファイルを選択していた場合は、正しいファイルを選択し、もう一度実行します。  
  
2.  **\[ファイルを開くアプリケーションの選択\]** ダイアログ ボックスで誤ってオブジェクト ブラウザーを選択した場合は、別のプログラムを選択してファイルを開きます。

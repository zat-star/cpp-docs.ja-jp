# &#39;With&#39; コンテキストおよびステートメントは、デバッグ ウィンドウでは有効ではありません
`With` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30726  
  
### このエラーを解決するには  
  
-   デバッグ コードから `With` ステートメントを削除します。

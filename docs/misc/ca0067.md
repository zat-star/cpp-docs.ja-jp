# CA0067
ディレクトリが見つかりません。  
  
 FxCopCmd コマンド ラインの **\/directory** オプションの値が見つかりません。  
  
 この警告は、[!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] の **CodeAnalysisDependentAssemblyPaths** プロパティを使用して追加の参照アセンブリの検索パスを指定したときに、それらのパスのいずれかが存在しないと発生する場合があります。  
  
-   警告 CA0067 が [CA0058](../misc/ca0058.md) エラーまたは [CA0060](../Topic/CA0060.md) 警告と共に表示された場合は、通常、そのもう一方のエラーを解決すると CA0067 警告も解決します。  
  
-   他のエラーまたは警告は示されずに警告 CA0067 のみが表示された場合、ディレクトリは不要であるため、この警告は無視してかまいません。

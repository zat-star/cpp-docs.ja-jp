# コマンド オプション、Visual Studio パッケージ ウィザード
このステップでは、Visual Studio の **\[ツール\]** メニューに表示されるコマンドの名前およびコマンド ID を定義します。  
  
 **コマンド名**  
 このボックスに表示されるテキストは、次のとおりです。  
  
-   コマンドの名前。  
  
-   ツール バーのコマンド ボタンのツールヒント。  
  
 **コマンド ID**  
 -   このボックスのテキストは、コマンドのコマンド ID を定義します。 この ID は、生成されたコードで使用され、このメニュー項目を参照する定数です。  
  
 **キャンセル**  
 -   ウィザードをキャンセルおよび停止します。  
  
 **前へ**  
 -   ウィザードの前のページに移動します。  
  
 **完了**  
 -   VSPackage を作成します。  
  
 **次へ**  
 -   ウィザードの次のページに移動します。

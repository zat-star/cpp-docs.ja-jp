# MSBuild エラー MSB1015
**MSBuild はこのバージョンのオペレーティング システムでは実行できません。  Windows 2000、Windows XP およびそれ以降のバージョンでのみサポートされています。**  
  
 [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] は、このバージョンのオペレーティング システムではサポートされません。  
  
### このエラーを解決するには  
  
1.  サポートされされている、いずれかのバージョンのオペレーティング システムに [!INCLUDE[vstecmsbuild](../build/includes/vstecmsbuild_md.md)] をインストールして実行してください。

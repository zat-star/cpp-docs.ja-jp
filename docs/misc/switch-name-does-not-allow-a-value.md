# Switch &lt;name&gt; does not allow a value.
このエラーが発生するのは一般的に、値を指定できないスイッチに値が割り当てられた場合です。  
  
### このエラーを解決するには  
  
1.  コマンドを入力し直して、スイッチの値は除きます。

# Item cannot be added to a project when multiple or no items are selected in Project Explorer.
このエラーが発生するのは一般的に、ソリューション エクスプローラーで項目が選択されていない場合、または複数の項目が選択されている場合です。  
  
### このエラーを解決するには  
  
1.  ソリューション エクスプローラーで項目を 1 つだけ選択します。

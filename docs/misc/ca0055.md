# CA0055
file:path を読み込むことができませんでした。\<\>  
  
 コード分析ツールは、分析用に指定された対象ファイルを読み込むことができませんでした。  これは、ファイルが見つからなかった場合、またはファイルへのアクセスが拒否された場合に発生します。  通常は、ユーザーが [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] の入力アセンブリ プロパティをオーバーライドすることを選択した場合、または fxcopcmd.exe を手動で呼び出した場合に発生します。

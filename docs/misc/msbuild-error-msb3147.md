# MSBuild エラー MSB3147
**MSB3147: 必要なファイル '\<ファイル\>' が '\<パス\>' で見つかりませんでした。**  
  
 このエラーは、ブートストラップがディスク上で setup.bin を見つけることができない場合に発生します。

# MSBuild エラー MSB3183
**MSB3183: 参照 '{アセンブリ}' は完全な信頼を必要とする相互運用機能アセンブリです。**  
  
 この警告は、部分的に信頼されたアプリケーションで使用するマニフェストの生成中に、参照に交ざって相互運用アセンブリが出現した場合に生成されます。

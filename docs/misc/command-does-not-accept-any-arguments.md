# Command does not accept any arguments.
このエラーが発生するのは一般的に、スイッチしか受け入れないコマンドの引数を入力した場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントでコマンドの正しい構文を確認して、やり直します。

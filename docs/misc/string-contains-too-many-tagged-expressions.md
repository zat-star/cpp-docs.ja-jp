# String contains too many tagged expressions.
このエラーが発生するのは一般的に、検索文字列で 9 つを超えるタグ付き正規表現が指定されている場合です。  
  
### このエラーを解決するには  
  
1.  指定するタグ付き正規表現の数を減らします。

# CA0503
プロパティの使用は推奨されていません。  優先されるプロパティを使用します。  
  
 警告 CA0503 は、使用が推奨されていないコード分析のプロパティのビルド構成ファイルまたはプロジェクト構成ファイルに値が含まれている場合に生成されます。  この警告を解決するには、指定された優先されるプロパティを使用してください。

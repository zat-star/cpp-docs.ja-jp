# 省略記号 (...)
省略記号演算子 \(`...`\) は、コンテキストに応じて C および C\+\+ で多くの用途に使用されています。  当初は C 言語の可変関数のパラメーター リストで抽象宣言子として使用されていましたが、現在は、プリプロセッサ マクロのプログラミングでも可変個引数マクロの形式で使用されます。  C\+\+ では `...` を `catch` の例外処理ブロックに使用し、C\+\+11 では可変個引数テンプレートに使用しています。

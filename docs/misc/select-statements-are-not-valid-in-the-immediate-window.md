# &#39;Select&#39; ステートメントは、イミディエイト ウィンドウでは有効ではありません
`Select` ステートメントは、ソース コードでのみ使用できます。  
  
 **エラー ID:** BC30718  
  
### このエラーを解決するには  
  
-   デバッグ コードから `Select` ステートメントを削除します。

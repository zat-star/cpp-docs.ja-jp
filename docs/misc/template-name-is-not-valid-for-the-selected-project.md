# Template &lt;name&gt; is not valid for the selected project.
このエラーが発生するのは一般的に、**New File** コマンドの `File.NewFile` または **Add New Solution Item** コマンドの `File.AddNewSolutionItem` に不明なテンプレート名を入力した場合です。  
  
### このエラーを解決するには  
  
1.  コマンドと引数を入力し直します。  
  
     または  
  
2.  New File コマンドまたは Add New Solution Item コマンドの正しい構文についてのトピックを確認します。

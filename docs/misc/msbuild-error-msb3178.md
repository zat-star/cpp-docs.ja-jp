# MSBuild エラー MSB3178
**MSB3178: アセンブリ '\<アセンブリ\>' はファイルとして正しく指定されていません。**  
  
 この警告は、アプリケーション マニフェストの生成中に、ビルド処理が、ファイル参照が実際には \(マネージまたはネイティブ\) アセンブリであることを検出した場合に生成されます。  
  
### このエラーを解決するには  
  
-   MSBuild エラーの原因と解決方法の詳細については、「[追加のリソース](../Topic/Additional%20MSBuild%20Resources.md)」を参照してください。

# リモート サーバー コンピューターが存在しないか、利用できません (Visual Basic)。
使用できないか、存在しないリモート サーバーに接続しようとしています。 このエラーは、ネットワーク接続が失われていることが原因である場合があります。  
  
### このエラーを解決するには  
  
-   ネットワークに再接続し、もう一度やり直してください。

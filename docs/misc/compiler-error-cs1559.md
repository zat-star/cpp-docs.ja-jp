# コンパイラ エラー CS1559
'object' はインポートされているため、Main メソッドに対して使うことはできません。  
  
 [\/main](../Topic/-main%20\(C%23%20Compiler%20Options\).md) コンパイラ オプションに無効なクラスが指定されています。このクラスは [Main](../Topic/Main\(\)%20and%20Command-Line%20Arguments%20\(C%23%20Programming%20Guide\).md) メソッドの場所として使用することはできません。

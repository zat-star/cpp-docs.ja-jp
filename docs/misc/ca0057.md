# CA0057
違反は現在のターゲットおよび規則のセットにマップできません。  
  
 現在のターゲットと規則のセットに割り当てられない違反がプロジェクト ファイルまたはインポート済み XML レポートに含まれているときに、FxCopCmd.exe が手動で起動された場合、このエラーが発生します。  
  
 コマンドライン バージョンの分析ツールを使用して分析を実行した場合にのみ発生します。

# Look In list must contain valid files or folders, separated by semicolons.
通常、このエラーが発生するのは、\[検索対象\] ボックスに表示されるディレクトリ名またはファイル名に無効な文字が含まれている場合です。  
  
### このエラーを解決するには  
  
1.  パスまたはファイル名を入力するのではなく、\[検索対象\] ダイアログ ボックスを使用して、ディレクトリまたはファイル名を検索します。  \[検索対象\] ダイアログ ボックスを開くには、\[検索対象\] ボックスの横の省略記号 \(...\) ボタンをクリックします。

# MSBuild エラー MSB3121
**MSB3121: アプリケーション マニフェスト内のファイルの関連付け要素に、必要な属性 \(拡張子、説明、progid、または既定のアイコン\) が 1 つ以上がありません。**  
  
 [ClickOnce 配置マニフェスト](../Topic/ClickOnce%20Deployment%20Manifest.md) には 4 つの属性の値がすべて含まれている必要があります。  
  
### このエラーを解決するには  
  
-   各 [ClickOnce 配置マニフェスト](../Topic/ClickOnce%20Deployment%20Manifest.md) 属性を有効な値に設定します。

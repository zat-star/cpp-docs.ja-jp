# The application cannot undo.
このエラーが発生するのは一般的に、別のファイルとリンクした取り消し操作があった項目に対して、元に戻す操作を行おうとする場合です。  リンクした変更の後に動作が発生した場合、Visual Studio は変更をロールバックできません。  
  
### このエラーを解決するには  
  
1.  変更を手動で元に戻します。

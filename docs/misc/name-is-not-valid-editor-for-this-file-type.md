# &lt;name&gt; is not valid editor for this file type.
このエラーが発生するのは一般的に、\[コマンド\] ウィンドウまたは \[検索\] ボックスで引数 \/e:*editorname* に入力した値が正しくない場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントで正しい \/e:*editorname* 構文を確認し、コマンドを入力し直します。

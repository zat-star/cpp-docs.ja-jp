# セキュリティ属性を &#39;&lt;name&gt;&#39; に適用できません: &lt;error&gt;
セキュリティ属性を適用するときにエラーが発生しました。 セキュリティ属性の使い方が間違っているか、コンパイラに問題がある可能性があります。  
  
 **エラー ID:** BC31520  
  
### このエラーを解決するには  
  
1.  属性の構文と引数を確認します。  
  
2.  プログラムをもう一度コンパイルし、エラーがまだ発生するかどうか確認します。  
  
3.  エラーが引き続き発生する場合は、作業内容を保存し、[!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)] を再起動します。  
  
4.  エラーがまだ発生する場合は、[!INCLUDE[vbprvb](../Token/vbprvb_md.md)] を再インストールします。  
  
5.  再インストールした後にエラーが続く場合は、マイクロソフト プロダクト サポート サービスに通知してください。

# リソース コンパイラの致命的なエラー RC1207
既定のコード ページが正しくありません  
  
 **RCCODEPAGE** 環境変数によって指定されたコード ページが正しくありません。 詳しくは、「*ランタイム ライブラリ リファレンス*」の「[コード ページ](../c-runtime-library/code-pages.md)」をご覧ください。

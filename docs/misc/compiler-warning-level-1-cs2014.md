# コンパイラの警告 (レベル 1) CS2014
コンパイラ オプション 'old option' は古い形式です。代わりに 'new option' を使用してください。  
  
 このコンパイラ オプション形式は使用されなくなりました。 詳細については、「[C\# コンパイラ オプション](../Topic/C%23%20Compiler%20Options.md)」を参照してください。

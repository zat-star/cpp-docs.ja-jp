# CA0504
規則ディレクトリは存在しないため無視されました  
  
 警告 CA504 は、プロジェクトまたはビルド構成ファイルから、**CodeAnalysisRuleDirectories** プロパティで指定されたディレクトリが見つからなかった場合に生成されます。  この問題を解決するには、有効なディレクトリ パスを入力します。

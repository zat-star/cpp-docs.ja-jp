# Command allows at most &lt;number&gt; arguments.
このエラーが発生するのは一般的に、コマンドに入力された引数が多すぎる場合、または引数の組み合わせが正しくない場合です。  
  
### このエラーを解決するには  
  
1.  ドキュメントでコマンドの正しい構文を確認して、やり直します。

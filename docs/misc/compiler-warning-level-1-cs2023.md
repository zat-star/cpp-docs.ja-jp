# コンパイラの警告 (レベル 1) CS2023
応答ファイルで指定されているため、\/noconfig オプションを無視します  
  
 [\/noconfig](../Topic/-noconfig%20\(C%23%20Compiler%20Options\).md) コンパイラ オプションが応答ファイルに指定されましたが、これは許可されていません。  
  
 この警告は [\/nowarn](../Topic/-nowarn%20\(C%23%20Compiler%20Options\).md) オプションで抑制できません。

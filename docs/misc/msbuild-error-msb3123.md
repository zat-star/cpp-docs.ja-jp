# MSBuild エラー MSB3123
**MSB3123: ファイルの関連付けの数が制限 \<limit\> を超えています。**  
  
 各ファイルに関連付けることのできるファイル名拡張子の数は制限されています。  このエラーは、対象のオペレーティング システムで許可されている数より多い数のファイル名拡張子を関連付けようとすると発生します。  
  
### このエラーを解決するには  
  
-   ファイルの関連付けの数を、指定されている数以下にします。

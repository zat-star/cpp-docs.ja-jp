# MSBuild エラー MSB4137
**MSB4137: "\<x\>" の構成ファイル内に指定されている値が無効です。  プロパティ名またはツールのバージョン名が空の文字列になっています。**  
  
 ツールセット定義に無効なプロパティ名または値が含まれています。  
  
### このエラーを解決するには  
  
-   ツールセット定義に無効または空のプロパティ名または値が含まれていないことを確認してください。

# Command was not specified.
このエラーが発生するのは一般的に、**\[コマンド\]** ウィンドウまたは **\[検索\]** ボックスに空の文字列を入力した場合です。  
  
### このエラーを解決するには  
  
1.  有効なコマンドを入力します。  詳細については、「[Visual Studio コマンド](../Topic/Visual%20Studio%20Commands.md)」を参照してください。

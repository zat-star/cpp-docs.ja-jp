# モジュール内の型を &#39;&lt;specifier&gt;&#39; として宣言することはできません
無効な修飾子を持つ型がモジュール内で宣言されました。  
  
 **エラー ID:** BC30735  
  
### このエラーを解決するには  
  
-   モジュール内で宣言した型に対して `Protected` 修飾子や `Protected Friend` 修飾子を使用しません。

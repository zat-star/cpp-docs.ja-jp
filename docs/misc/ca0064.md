# CA0064
指定された規則セットに FxCop 規則が含まれていなかったため、分析は行われませんでした。  
  
 この警告は、次のいずれかの状況で発生する場合があります。  
  
-   この警告が CA0063 の警告と共に表示された場合は、規則セット ファイルの読み込みで問題が発生しました。  詳細については、「[CA0063](../misc/ca0063.md)」を参照してください。  
  
-   このエラーが CA0062 の警告と共に表示された場合は、規則セットで指定された規則を含むアセンブリが見つからなかった可能性があります。  詳細については、「[CA0062](../Topic/CA0062.md)」を参照してください。  
  
-   その他の場合、この警告は通常、規則セットが空か、子の規則セットで有効になっている規則がすべて無効になっている場合に発生します。  [!INCLUDE[vs_current_short](../misc/includes/vs_current_short_md.md)] の規則セット エディターを使用して、規則セットのいくつかの規則を有効にしてください。  詳細については、「[カスタム規則セットの作成](../Topic/Creating%20Custom%20Code%20Analysis%20Rule%20Sets.md)」を参照してください。

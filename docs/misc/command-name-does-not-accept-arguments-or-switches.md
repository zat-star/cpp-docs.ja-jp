# Command &lt;name&gt; does not accept arguments or switches.
このエラーが発生するのは一般的に、引数やスイッチを使用しないコマンドまたはエイリアスで、スイッチまたは引数を入力した場合です。  エイリアスを入力した場合は、エラー メッセージのテキストにはそのエイリアスではなく完全なコマンド名が表示されます。  スイッチまたは引数を使用する有効なコマンドの一覧については、「[Visual Studio コマンド](../Topic/Visual%20Studio%20Commands.md)」を参照してください。  
  
### このエラーを解決するには  
  
1.  コマンドと引数を入力し直すか、新しいコマンドを入力します。

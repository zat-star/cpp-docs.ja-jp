# Command does not accept switches.
このエラーが発生するのは一般的に、引数しか認識しないコマンドにスイッチを指定した場合です。  
  
### このエラーを解決するには  
  
1.  コマンドの構文要件を再確認します。  詳細については、「[引数を使用する Visual Studio コマンド](../Topic/Visual%20Studio%20Commands.md)」を参照してください。

# piecewise_constant_distribution::densities
格納されている分布密度を返します。  
  
## 構文  
  
```  
std::vector<double> densities() const;  
```  
  
## 解説  
 メンバー関数は、重みパラメーターから計算された、格納されている密度を返します。  
  
## 必要条件  
 **ヘッダー:** \<random\>  
  
 **名前空間:** std

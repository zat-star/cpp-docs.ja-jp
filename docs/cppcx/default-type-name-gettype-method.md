# default::(type_name)::GetType メソッド
現在の型を表す Platform::Type^ を返します。  
  
## 構文  
  
```cpp  
  
Platform::Type^ GetType()  
```  
  
## 戻り値  
 現在のオブジェクトを表す [Platform::Type](../cppcx/platform-type-class.md)^ オブジェクト。

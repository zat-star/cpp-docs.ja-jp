# <a name="deleting-open-files"></a>開いているファイルの削除
**ANSI 4.9.4.1** 開いているファイルへの削除関数の効果  
  
 削除関数はファイルを削除します。 ファイルが開いている場合、この関数は失敗し、-1 を返します。

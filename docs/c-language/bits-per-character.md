# 文字あたりのビット数
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 2.2.4.2.1** 文字のビット数  
  
 文字のビット数は、マニフェスト定数 **CHAR\_BIT** によって表されます。  LIMITS.H ファイルは **CHAR\_BIT** を 8 として定義します。

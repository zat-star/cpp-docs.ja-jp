# <a name="bits-per-character"></a>Bits per Character (文字あたりのビット数)
**ANSI 2.2.4.2.1** 文のビット数  
  
 文字のビット数は、マニフェスト定数 **CHAR_BIT** によって表されます。 LIMITS.H ファイルは **CHAR_BIT** を 8 として定義します。

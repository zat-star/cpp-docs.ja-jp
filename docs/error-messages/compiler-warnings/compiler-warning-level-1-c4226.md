# <a name="compiler-warning-level-1-c4226"></a>コンパイラの警告 (レベル 1) C4226

> 使用される標準の拡張機能: '*キーワード*' 残されているキーワード

現在のバージョンの Visual C では、このキーワードは使用しません。

この警告は、自動的にエラーになります。

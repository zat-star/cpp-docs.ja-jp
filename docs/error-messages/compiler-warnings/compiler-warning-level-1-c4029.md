# <a name="compiler-warning-level-1-c4029"></a>コンパイラの警告 (レベル 1) C4029
宣言された仮パラメーター リストの定義と異なります  
  
 関数宣言の仮パラメーターの型が関数の定義と一致しません。 コンパイラは、定義のパラメーター リストを使用します。

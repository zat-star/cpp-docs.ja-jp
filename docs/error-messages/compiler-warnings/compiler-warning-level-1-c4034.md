# <a name="compiler-warning-level-1-c4034"></a>コンパイラの警告 (レベル 1) C4034
sizeof は 0 を返します  
  
 `sizeof`サイズが 0 のオペランドに演算子を適用 (空の構造体、共用体、クラス、または列挙型、または型`void`)。

# コンパイラの警告 (レベル 1) C4340
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'value' : 正の値から負の値へラップしました。  
  
 `enum` 値が、`enum` 値に設定できる最大の正の数より大きいため、負の値にラップされました。

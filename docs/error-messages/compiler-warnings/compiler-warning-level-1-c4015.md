# コンパイラの警告 (レベル 1) C4015
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : ビット フィールドは、整数型として宣言されていません。  
  
 ビット フィールドが整数型として宣言されていません。  コンパイラは、ビット フィールドの基本型を unsigned と見なします。  ビット フィールドは符号なしの整数型として宣言する必要があります。

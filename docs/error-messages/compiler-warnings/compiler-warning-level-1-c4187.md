# <a name="compiler-warning-level-1-c4187"></a>コンパイラの警告 (レベル 1) C4187
\#インポート属性 'attribute1' と 'attribute2' は互換性がありません。両方とも無視されます。  
  
 A [#import](../../preprocessor/hash-import-directive-cpp.md)指定されたステートメント`no_implementation`と`implementation_only`属性です。 両方とも無視されます。

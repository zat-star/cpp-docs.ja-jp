# <a name="compiler-warning-c4961"></a>コンパイラの警告 C4961
プロファイル データが '.pgd ファイル' にマージされませんでした。プロファイルのガイド付き最適化を無効にします  
  
 プロファイル データ (.pgc ファイル) がなかったため、プロファイルのガイド付き最適化を実行できません。

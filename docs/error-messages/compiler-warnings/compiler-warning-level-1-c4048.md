# <a name="compiler-warning-level-1-c4048"></a>コンパイラの警告 (レベル 1) C4048
別の宣言された配列の添字: 'identifier1' と 'identifier2'  
  
 式には、異なるサイズの配列へのポインターが含まれます。 ポインターは変換されずに使用されます。  
  
 同じまたは同等の型への配列を明示的にキャストする場合、この警告を解決可能性があります。

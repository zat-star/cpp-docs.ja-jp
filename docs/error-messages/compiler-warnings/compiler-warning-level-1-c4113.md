# <a name="compiler-warning-level-1-c4113"></a>コンパイラの警告 (レベル 1) C4113
'identifier1' と 'identifier2' のパラメーター リスト  
  
 関数ポインターが別の関数ポインターに割り当てられているが、関数の仮パラメーター リストが一致しません。 割り当ては変更されずにコンパイルします。

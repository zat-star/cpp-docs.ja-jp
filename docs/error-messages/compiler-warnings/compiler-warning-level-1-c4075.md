# <a name="compiler-warning-level-1-c4075"></a>コンパイラの警告 (レベル 1) C4075
初期化子が不明な初期化領域にあります。  
  
 [#pragma init_seg](../../preprocessor/init-seg.md) が認識できないセクション名を使用しています。 コンパイラはこの **pragma** コマンドを無視します。  
  
 次の例では C4075 が生成されます。  
  
```  
// C4075.cpp  
// compile with: /W1  
#pragma init_seg("mysegg")   // C4075  
  
// try..  
// #pragma init_seg(user)  
  
int main() {  
}  
```

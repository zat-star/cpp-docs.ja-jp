# <a name="compiler-warning-level-1-c4158"></a>コンパイラの警告 (レベル 1) C4158
#pragma pointers_to_members(full_generality, inheritance) と仮定します。  
  
 A **#pragma pointers_to_members (***単一 | 複数 | 仮想***)** 、付随するなしに発行されて**#pragma (full_generality)**します。

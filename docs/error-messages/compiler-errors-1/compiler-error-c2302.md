# <a name="compiler-error-c2302"></a>コンパイラ エラー C2302
'.~identifier' の左側はクラス、構造体、共用体型でなければなりません  
  
 ピリオド (.) 演算子の左側の式はクラス、構造体、または共用体ではありません。

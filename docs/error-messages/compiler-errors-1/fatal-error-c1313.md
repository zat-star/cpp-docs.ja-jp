# 致命的なエラー C1313
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コンパイラ制限 : '型' ブロックは、'数値' レベルよりも深く入れ子にすることはできません。  
  
 例外処理または構造化例外処理のブロックを入れ子していますが、入れ子のレベルが深すぎます。コードを簡潔にしてください。

# <a name="compiler-error-c2310"></a>コンパイラ エラー C2310
catch ハンドラーは 1 つの型を指定する必要があります。  
  
 Catch ハンドラーは、型がないか、複数の種類を指定します。  
  
 次の例では、C2310 が生成されます。  
  
```  
// C2310.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   try {  
      throw "Out of memory!";  
   }  
   catch( int ,int) {}   // C2310 two types  
   // try the following line instead  
   // catch( int)  {}  
}  
```

# コンパイラ エラー C2172
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 実パラメーターがポインターではありません : パラメーター番号  
  
 パラメーター `number` はポインターではありません。 この関数にはポインターが必要です。

# <a name="fatal-error-c1100"></a>致命的なエラー C1100
OLE を初期化できませんシステム エラー メッセージ。  
  
 コンパイラは、コンポーネント オブジェクト モデル (COM) ライブラリを初期化できません。 参照してください[CoInitialize](http://msdn.microsoft.com/library/windows/desktop/ms678543)です。

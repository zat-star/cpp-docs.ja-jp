# <a name="compiler-error-c2063"></a>コンパイラ エラー C2063
'identifier': 関数ではありません。  
  
 識別子が関数として使用されていますが、関数として宣言されていません。  
  
 次の例では C2063 エラーが生成されます。  
  
```  
// C2063.c  
int main() {  
   int i, j;  
   j = i();    // C2063, i is not a function  
}  
```  
  
 考えられる解決策:  
  
```  
// C2063b.c  
int i() { return 0;}  
int main() {  
   int j;  
   j = i();  
}  
```

# <a name="fatal-error-c1064"></a>致命的なエラー C1064
コンパイラの制限: トークンには、内部バッファーがオーバーフローしました。  
  
 識別子は、識別子に使用する内部バッファーの長さを超えています。 名前を短くしてください。

# <a name="compiler-error-c2095"></a>コンパイラ エラー C2095
'function': 実際のパラメーターが型 'void': パラメーター 'number'  
  
 関数に渡されるパラメーターの型が`void`、これは許可されません。 Void へのポインターを使用して ( `void *`) 代わりにします。  
  
 `number`パラメーターが示す`void`です。

# 致命的なエラー C1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コンパイラの制限 : マクロの入れ子のレベルが深すぎます。  
  
 同時に展開を試みたマクロの数が多すぎます。  マクロの入れ子レベルは 256 までに制限されています。  入れ子になったマクロを簡単なマクロに分割してください。

# コンパイラ エラー C2349
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

 詳細については、「[コンパイラの警告 \(レベル 1 および 3\) C4793](../../error-messages/compiler-warnings/compiler-warning-level-1-and-3-c4793.md)」を参照してください。

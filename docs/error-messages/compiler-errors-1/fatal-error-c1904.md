# <a name="fatal-error-c1904"></a>致命的なエラー C1904

 このエラーは、属性プロバイダーのエラーを示します。  
  
 エラーの状況を再現しようとして問題を特定し、再現可能なテスト_ケースを作成しにお問い合わせください[ご意見](/visualstudio/ide/talk-to-us)します。

# <a name="compiler-error-c2313"></a>コンパイラ エラー C2313
'type1': 行番号の参照 ('type2') でキャッチされます  
  
 例外の型には 2 つのハンドラーがあります。 2 番目の catch の型は、最初の型への参照です。  
  
 次の例では C2313 が生成されます。  
  
```  
// C2313.cpp  
// compile with: /EHsc  
#include <eh.h>  
class C {};  
int main() {  
    try {  
        throw "ooops!";  
    }  
    catch( C& ) {}  
    catch( C ) {}   // C2313  
}  
```

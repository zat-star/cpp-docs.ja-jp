# コンパイラ エラー C2185
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : \_based 割り当てが無効です。  
  
 レジスタ変数または自動 \(ローカル\) 変数が `__based` として宣言されました。  `__based` として宣言できるのは、グローバル変数だけです。

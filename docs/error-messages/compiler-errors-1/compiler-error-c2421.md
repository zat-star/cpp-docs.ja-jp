# コンパイラ エラー C2421
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

PTR 演算子は、'context' のレジスタ オペランドと一緒に使えません。  
  
 `PTR` 演算子は、`register` オペランドには使用できません。

# コンパイラ エラー C2300
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : クラスに '~identifier' というデストラクターがありません  
  
 このクラスには、必要な名前のデストラクターがありません。

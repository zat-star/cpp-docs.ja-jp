# コンパイラ エラー C2406
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 'context' で名前が定義されていません  
  
 未定義の識別子が `SIZE`、`LENGTH`、またはメンバー選択 \(.\) 演算子で使用されています。

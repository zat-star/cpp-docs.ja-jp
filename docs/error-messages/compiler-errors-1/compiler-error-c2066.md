# コンパイラ エラー C2066
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

関数型へのキャストが正しくありません。  
  
 ANSI C では、関数ポインターとデータ ポインターの間でのキャストは無効です。

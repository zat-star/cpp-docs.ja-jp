# コンパイラ エラー C2020
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'member' : 'class' のメンバーが再定義されています。  
  
 基本クラスまたは構造体から継承されたメンバーが再定義されています。  継承されたメンバーは、基本クラスで `virtual` として宣言されない限り再定義できません。

# <a name="compiler-error-c2145"></a>コンパイラ エラー C2145
構文エラー: 識別子の前に 'token' が必要です。  
  
 コンパイラでは `token` が必要でしたが、識別子が検出されました。  
  
 このエラーは、ブロック内の最後の宣言の後にセミコロンがないと、発生する可能性があります。

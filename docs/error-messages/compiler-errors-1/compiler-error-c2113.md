# コンパイラ エラー C2113
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'\-' : ポインター以外からポインターを減算することはできません。  
  
 減算演算の右オペランドはポインターですが、左オペランドはポインターではありません。

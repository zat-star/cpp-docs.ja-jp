# <a name="fatal-error-c1065"></a>致命的なエラー C1065
コンパイラの制限: タグが足りません  
  
 ソース ファイルには、複数の 65,523 のクラス、構造体、共用体、名前空間、または列挙型が含まれています。  
  
 ファイルを小さなファイルに分割します。

# コンパイラ エラー C2205
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier' : 外部変数は、ブロック スコープ内では初期化できません。  
  
 `extern` 変数は関数内では初期化できません。

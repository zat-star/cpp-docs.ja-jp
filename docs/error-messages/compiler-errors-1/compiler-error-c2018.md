# <a name="compiler-error-c2018"></a>コンパイラ エラー C2018
文字 'hexnumber' は認識できません  
  
 ソース ファイルには、予期しない、16 進の番号によって識別される ASCII 文字が含まれています。 エラーを解決するのには、文字を削除します。

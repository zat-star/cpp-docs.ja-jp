# <a name="compiler-error-c2181"></a>コンパイラ エラー C2181
else 文が if と一致しません。  
  
 各 `else` には一致する `if`が必要です。  
  
 次の例では C2181 が生成されます。  
  
```  
// C2181.cpp  
int main() {  
   int i = 0;  
   else   // C2181  
      i = 1;  
}  
```  
  
 考えられる解決策:  
  
```  
// C2181b.cpp  
int main() {  
   int i = 0;  
   if(i)  
      i = 0;  
   else  
      i = 1;  
}  
```

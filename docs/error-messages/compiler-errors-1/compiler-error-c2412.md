# コンパイラ エラー C2412
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'label' : ラベルが、現在の関数内で 2 回以上定義されました。  
  
 ラベルが現在の関数内で 2 回以上定義されています。  ラベル名とそのラベルを参照している箇所のラベル名を変更してください。

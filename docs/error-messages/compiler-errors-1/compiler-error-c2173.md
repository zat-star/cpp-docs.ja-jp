# <a name="compiler-error-c2173"></a>コンパイラ エラー C2173
'function' : 実引数がポインターではありません : パラメーター番号 1、パラメーター リスト番号 2  
  
 パラメーター リスト `number1` に渡されたパラメーター `number2` はポインターではありません。 この関数にはポインターが必要です。

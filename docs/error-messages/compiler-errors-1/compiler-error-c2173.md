# コンパイラ エラー C2173
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : 実引数がポインターではありません : パラメーター番号 1、パラメーター リスト番号 2  
  
 パラメーター リスト `number2` に渡されたパラメーター `number1` はポインターではありません。 この関数にはポインターが必要です。

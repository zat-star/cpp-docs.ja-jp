# コンパイラ エラー C2105
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator' には左辺値が必要です。  
  
 この演算子のオペランドには左辺値が必要です。  
  
 次の例では警告 C2105 が生成されます。  
  
```  
// C2105.cpp  
int main() {  
   unsigned char * p1 = 0;  
   unsigned int * p2 = (unsigned int *)p1;  
   p2++;  
  
   unsigned int * p = 0;  
   p++;   // ok  
  
   p2 = (unsigned int *)p1;  
   ((unsigned int *)p1)++;   // C2105  
}  
```  
  
 次の例では警告 C2105 が生成されます。  
  
```  
// C2105b.cpp  
int main() {  
   int a[10] = {0};  
   int b[10] = {0};  
   ++(a , b);   // C2105  
  
   // OK  
   ++(a[0] , b[0]);  
   ++a[0];  
}  
```

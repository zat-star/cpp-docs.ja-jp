# <a name="compiler-error-c2224"></a>コンパイラ エラー C2224
'.identifier' の左側が構造体/共用体型をいる必要があります。  
  
 ピリオド (.) の左側のオペランドはクラス、構造体、または共用体ではありません。  
  
 このエラーは、未定義の変数は、左のオペランドで発生することができます (したがって入力`int`)。

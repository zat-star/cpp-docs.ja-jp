# コンパイラ エラー C2043
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'break' が正しくありません。  
  
 [break](../../cpp/break-statement-cpp.md) は、`do`、`for`、`while`、または `switch` ステートメント内でのみ有効です。

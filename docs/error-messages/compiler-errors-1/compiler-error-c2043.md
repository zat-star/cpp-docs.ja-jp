# <a name="compiler-error-c2043"></a>コンパイラ エラー C2043
'break' が正しくありません。  
  
 [break](../../cpp/break-statement-cpp.md) は、 `do`、 `for`、 `while`、または `switch` ステートメント内でのみ有効です。

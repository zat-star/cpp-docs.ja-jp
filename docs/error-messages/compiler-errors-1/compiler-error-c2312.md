# <a name="compiler-error-c2312"></a>コンパイラ エラー C2312
'exception1': 行番号の 'exception2' でキャッチされました  
  
 2 つのハンドラーが同じ例外型をキャッチします。  
  
 次の例では C2312 が生成されます。  
  
```  
// C2312.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
    try {  
        throw "ooops!";  
    }  
    catch( signed int ) {}  
    catch( int ) {}   // C2312  
}  
```

# コンパイラ エラー C2138
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

空の列挙定数が定義されました。  
  
 \/Za \(Microsoft 拡張機能の無効化\) オプションを指定した場合、列挙定数には少なくとも 1 つのメンバーが必要です。

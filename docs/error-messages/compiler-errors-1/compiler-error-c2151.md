# <a name="compiler-error-c2151"></a>コンパイラ エラー C2151
1 つ以上の language 属性  
  
 関数が 1 つ以上のキーワードを持つ ( `__cdecl`、 `__stdcall`、または`__fastcall`) 呼び出し規則を指定します。

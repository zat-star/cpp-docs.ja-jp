# <a name="fatal-error-c1351"></a>致命的なエラー C1351
dll 'dll' の読み込みエラー: 互換性のないバージョン  
  
 間違ったバージョンの DLL が見つかりました。  
  
 これは、インストールに問題を示し、製品 CD から DLL を再インストールする必要があります。

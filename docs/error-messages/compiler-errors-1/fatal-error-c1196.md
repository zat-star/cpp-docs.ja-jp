# <a name="fatal-error-c1196"></a>致命的なエラー C1196
'identifier': 型ライブラリ 'typelib' で見つかった識別子は、有効な C++ 識別子ではありません  
  
 型ライブラリのいずれかの識別子が有効な C++ 識別子ではありません。 使用可能なタイプ ライブラリは[#import](../../preprocessor/hash-import-directive-cpp.md)です。

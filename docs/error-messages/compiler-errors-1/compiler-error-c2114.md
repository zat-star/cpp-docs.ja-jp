# <a name="compiler-error-c2114"></a>コンパイラ エラー C2114
'operator' : 左オペランドがポインターなので、右オペランドは整数値でなければなりません  
  
 `operator` の左オペランド がポインターなので、右オペランドは整数値でなければなりません。

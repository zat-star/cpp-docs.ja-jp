# <a name="fatal-error-c1121"></a>致命的なエラー C1121
CryptoAPI への呼び出しに失敗しました  
  
 コンパイラによって、CryptoAPI への呼び出しと呼び出しに失敗しました。 Visual Studio を再インストールし、オペレーティング システムの再インストールします。

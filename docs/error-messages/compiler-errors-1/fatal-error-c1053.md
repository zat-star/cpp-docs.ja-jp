# <a name="fatal-error-c1053"></a>致命的なエラー C1053
'\<識別子 >': 関数が大きすぎます  
  
 コンパイルするには関数が大きすぎます。  
  
### <a name="to-fix-by-using-the-following-possible-solutions"></a>以下の可能性がある解決策を使って修正するには  
  
1.  最適化せずにコンパイルしてみてください。  
  
2.  大きな関数を小さな関数に分割します。  
  
3.  インライン関数の呼び出しを減らします。

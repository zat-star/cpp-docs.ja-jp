# コンパイラ エラー C2368
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier': 再定義されています。異なるアロケーション指定子です  
  
 シンボルの宣言と定義で、異なる `__declspec` 属性を指定しています。

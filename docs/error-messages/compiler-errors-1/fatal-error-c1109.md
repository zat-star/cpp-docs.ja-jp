# <a name="fatal-error-c1109"></a>致命的なエラー C1109
DLL ファイル 'dll' に 'entry point' が見つかりません  
  
 エントリ ポイントで、[遅延読み込みされた DLL](../../build/reference/linker-support-for-delay-loaded-dlls.md)で必要なコンパイラが見つかりませんでした。

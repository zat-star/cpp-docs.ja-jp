# <a name="compiler-error-c2913"></a>コンパイラ エラー C2913
明示的な特殊化です。'declaration' はクラス テンプレートの特殊化ではありません。  
  
 非テンプレート クラスを特化することはできません。  
  
 次の例では、C2913 が生成されます。  
  
```  
// C2913.cpp  
// compile with: /c  
class X{};  
template <class T> class Y{};  
  
template<> class X<int> {};   // C2913  
template<> class Y<int> {};  
```

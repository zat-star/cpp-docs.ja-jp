# <a name="compiler-error-c3874"></a>コンパイラ エラー C3874
'function' の戻り値の型が 'type' の代わりに ' int' にする必要があります。  
  
 関数には、コンパイラが必要とした戻り値の型はありません。  
  
 次の例では、c3874 エラーが生成されます。  
  
```  
// C3874b.cpp  
double main()  
{   // C3874  
}  
```

# コンパイラ エラー C2526
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identifier1' : C のリンケージ関数は C\+\+ のクラス 'identifier2' を戻せません。  
  
 C のリンケージで定義した関数からユーザー定義型を返すことはできません。

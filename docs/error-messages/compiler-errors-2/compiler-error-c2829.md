# <a name="compiler-error-c2829"></a>コンパイラ エラー C2829
'operator 演算子' は、変数パラメーター リストを持つことはできません。  
  
 2 つの演算子は、変数パラメーター リストを実行できます。  
  
-   関数呼び出し`()`  
  
-   `new`

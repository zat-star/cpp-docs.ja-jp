# <a name="compiler-error-c2648"></a>コンパイラ エラー C2648
'identifier': 静的メンバーが既定のパラメーターとしてメンバーの使用  
  
 非静的メンバーは、既定のパラメーターとして使用されます。  
  
 次の例では、C2648 が生成されます。  
  
```  
// C2648.cpp  
// compile with: /c  
class C {  
public:  
   int i;  
   static int j;  
   void func1( int i = i );  // C2648  i is not static  
   void func2( int i = j );  // OK  
};  
```

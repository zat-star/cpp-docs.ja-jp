# <a name="compiler-error-c3501"></a>コンパイラ エラー C3501
ProgID 'progid' の登録された typelib はありません。  
  
 クラス ID を指定された progid が関連付けられているタイプ ライブラリ。 したがって、することはできませんに渡すことを progid、`#import`ステートメントです。

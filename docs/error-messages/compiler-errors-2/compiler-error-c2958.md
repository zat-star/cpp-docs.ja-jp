# <a name="compiler-error-c2958"></a>コンパイラ エラー C2958
'location' にある左側の区切り文字が正しく一致しませんでした。  
  
 区切り記号の整合がとれていません。  
  
 このエラーは、関数テンプレート宣言の一致しないかっこが発生することができます。

# <a name="compiler-error-c2958"></a>コンパイラ エラー C2958
左側の 'location' で見つかった区切り文字が正しく対応していません  
  
 区切り文字が正しく一致していません。  
  
 このエラーは、関数テンプレートの宣言に一致しないかっこが発生することができます。

# コンパイラ エラー C2817
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'operator delete' のリターン値の型は 'void' 型でなければなりません。  
  
 オーバーロードされた [operator delete](../Topic/operator%20delete%20\(%3Cnew%3E\).md) 関数は値を返すことができません。

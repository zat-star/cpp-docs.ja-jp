# <a name="compiler-error-c2817"></a>{1&gt;コンパイラ エラー C2817&lt;1}
'operator delete' の戻り値の型は 'void' 型でなければなりません。  
  
オーバー ロードされた[演算子 delete](../../standard-library/new-operators.md#op_delete)関数が値を返すことはできません。

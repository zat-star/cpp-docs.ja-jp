# <a name="compiler-error-c2794"></a>コンパイラ エラー C2794
'function': 'class' の直接または間接基底クラスのメンバーではありません  
  
 使用しようとしています。 [super](../../cpp/super.md)存在しないメンバー関数を呼び出す。  
  
 次の例には、C2794 が生成されます。  
  
```  
// C2794.cpp  
struct B {  
   void mf();  
};  
  
struct D : B {  
   void mf() {  
      __super::f();  // C2794  
   }  
};  
```

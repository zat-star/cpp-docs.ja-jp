# <a name="compiler-error-c2875"></a>コンパイラ エラー C2875
使用して宣言をにより、':identifier' の複数の宣言  
  
 宣言は、2 回定義する同じ項目です。  
  
 次の例では、c2875 エラーが生成されます。  
  
```  
// C2875.cpp  
struct A {  
   void f(int*);  
};  
  
struct B {  
   void f(double*);  
};  
  
struct AB : A, B {  
   using A::f;  
   using A::f;   // C2875  
   using B::f;  
};  
```

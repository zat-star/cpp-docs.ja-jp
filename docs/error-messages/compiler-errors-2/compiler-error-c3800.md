# <a name="compiler-error-c3800"></a>コンパイラ エラー C3800
'宣言': プロパティおよびイベントを混在させることはできません  
  
 構成要素は、プロパティとイベントの両方を宣言することはできません。  
  
 C3800 は古いコンパイラ オプションを使用して到達のみ**/clr:oldSyntax**します。

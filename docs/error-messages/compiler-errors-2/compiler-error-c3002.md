# <a name="compiler-error-c3002"></a>コンパイラ エラー C3002
'name1 name2' : 複数の OpenMP ディレクティブ名が存在します  
  
 複数のディレクティブ名を指定することはできません。  
  
 次の例では C3002 が生成されます。  
  
```  
// C3002.c  
// compile with: /openmp  
int main()  
{  
   #pragma omp parallel single   // C3002  
}  
```

# <a name="compiler-error-c3632"></a>コンパイラ エラー C3632

 [_ _event](../../cpp/event.md)宣言は、すべての構成要素では無効です。  
  
 C3632 は古いコンパイラ オプションを使用して到達のみ**/clr:oldSyntax**します。

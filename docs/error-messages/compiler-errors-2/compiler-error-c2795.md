# コンパイラ エラー C2795
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'super::function' はメンバー関数ではありません。  
  
 このエラー メッセージが表示されるのは、[super](../../cpp/super.md) を使用して、メンバー関数以外のメンバーにアクセスした場合です。

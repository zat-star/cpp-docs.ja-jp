# <a name="compiler-error-c2549"></a>コンパイラ エラー C2549
ユーザー定義の変換は、戻り値の型を指定できません。  
  
 次の例では、C2549 が生成されます。  
  
```  
// C2549.cpp  
// compile with: /c  
class X {  
public:  
   int operator int() { return value; }   // C2549  
  
   // try the following line instead  
   // operator int() { return value; }  
private:  
   int value;  
};  
```

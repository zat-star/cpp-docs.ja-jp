# コンパイラ エラー C2651
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'data type' : 'operator' の左側はクラス、構造体または共用体でなければなりません。  
  
 テンプレート パラメーターをクラスとして使用するには、整数型ではなくクラスでクラス テンプレートを特殊化します。

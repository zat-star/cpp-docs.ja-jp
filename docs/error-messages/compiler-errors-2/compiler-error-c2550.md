# <a name="compiler-error-c2550"></a>コンパイラ エラー C2550
'identifier': コンス トラクター初期化子リストはコンス トラクターの定義だけに許されます。  
  
 基底クラスの初期化子リストは、コンス トラクターではない関数の定義で使用されます。  
  
 次の例では、C2550 が生成されます。  
  
```  
// C2550.cpp  
// compile with: /c  
class C {  
public:  
   C();  
};  
  
class D : public C {  
public:  
   D();  
   void func();  
};  
  
void D::func() : C() {}  // C2550  
D::D() : C() {}   // OK  
```

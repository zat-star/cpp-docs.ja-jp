# <a name="compiler-error-c2557"></a>コンパイラ エラー C2557
'identifier': プライベート メンバー、プロテクト メンバーの初期化にはコンストラクターが必要です  
  
 プライベート メンバーまたはプロテクト メンバーに値を割り当てることができるのは、メンバーとフレンドのみです。 非パブリック メンバーは、クラス コンストラクターで初期化する必要があります。

# <a name="compiler-error-c3619"></a>コンパイラ エラー C3619
マネージ型または WinRT 型でテンプレートは宣言できません  
  
クラス テンプレートは、マネージ クラスや WinRT クラスまたはインターフェイスでは使用できません。  
  
C3619 は古いコンパイラ オプションを使用して到達のみ**/clr:oldSyntax**します。

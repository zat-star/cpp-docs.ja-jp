# <a name="compiler-error-c2634"></a>コンパイラ エラー C2634
'< class::member': 参照メンバーへのポインターは無効です  
  
 Reference メンバーへのポインターを宣言するとします。  
  
 次の例では、c2634 エラーが生成されます。  
  
```  
// C2634.cpp  
int mem;  
struct S {  
   S() : rf(mem) { }  
   int &rf;  
};  
int (S::*pdm) = &S::rf;   // C2634  
```

# <a name="compiler-error-c2809"></a>コンパイラ エラー C2809
'operator 演算子' に仮パラメーターがありません。  
  
 演算子には、必要なパラメーターが不足しています。  
  
 次の例では、C2809 が生成されます。  
  
```  
// C2809.cpp  
// compile with: /c  
class A{};  
int operator+ ();   // C2809  
int operator+ (A);   // OK  
```

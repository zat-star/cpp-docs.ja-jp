# <a name="compiler-error-c2655"></a>コンパイラ エラー C2655
'identifier': 現在のスコープに無効な宣言あるいは再定義  
  
 識別子は、グローバル スコープでのみ再宣言できます。  
  
 次の例では、C2655 が生成されます。  
  
```  
// C2655.cpp  
class A {};  
class B {  
public:  
   static int i;  
};  
  
int B::i;  // OK  
  
int main() {  
   A B::i;  // C2655  
}  
```

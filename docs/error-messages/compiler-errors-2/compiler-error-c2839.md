# コンパイラ エラー C2839
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

オーバーロードされた演算子「に対して無効な戻り値の型「型」\-\>」  
  
 `->` 演算子は、クラス、構造体、または共用体を返すか、クラス、構造体、または共用体への参照を返す必要があります。

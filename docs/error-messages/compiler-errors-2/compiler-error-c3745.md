# コンパイラ エラー C3745
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'function' : イベントだけが 'raised' することができます。  
  
 [\_\_raise](../../cpp/raise.md) キーワードに渡すことができるのは、[\_\_event](../../cpp/event.md) キーワードで定義された関数だけです。  
  
 次の例では警告 C3745 が生成されます。  
  
```  
// C3745.cpp  
struct E {  
   __event void func();  
   void func(int) {  
   }  
  
   void func2() {  
   }  
  
   void bar() {  
      __raise func();  
      __raise func(1);   // C3745  
      __raise func2();   // C3745  
   }  
};  
  
int main() {  
   E e;  
   __raise e.func();  
   __raise e.func(1);   // C3745  
   __raise e.func2();   // C3745  
}  
```

# <a name="compiler-error-c3277"></a>コンパイラ エラー C3277
マネージ '型' 内部 'enum' アンマネージ列挙体を定義することはできません。  
  
 列挙型が正しく定義されていないマネージ型の中。  
  
 次の例では、C3277 が生成されます。  
  
```  
// C3277a.cpp  
// compile with: /clr  
ref class A  
{  
   enum E {e1,e2};   // C3277  
   // try the following line instead  
   // enum class E {e1,e2};  
};  
  
int main()  
{  
}  
```

# <a name="compiler-error-c3166"></a>コンパイラ エラー C3166
'ポインター': 'type' のメンバーとして内部 _ _gc ポインターへのポインターを宣言することはできません  
  
無効なポインター宣言が見つかりました (、`__nogc`へのポインター、`__gc`ポインターです。)。 
  
C3166 は古いコンパイラ オプションを使用して到達のみ**/clr:oldSyntax**します。

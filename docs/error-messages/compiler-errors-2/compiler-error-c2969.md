# <a name="compiler-error-c2969"></a>コンパイラ エラー C2969
構文エラー: 'symbol': メンバー関数の定義を終了するための '}' がありません。  
  
 テンプレート メンバー関数の定義内に一致しないかっこがあります。  
  
 次の例では C2969 が生成されます。  
  
```  
// C2969.cpp  
// compile with: /c  
class A {  
   int i;  
public:  
   A(int i) {}  
};  
  
A anA(1);  
  
class B {  
   A a;  
   B() : a(anA);   // C2969  
   // try the following line instead  
   // B() : a(anA) {}  
};  
```

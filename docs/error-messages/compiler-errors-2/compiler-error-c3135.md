# <a name="compiler-error-c3135"></a>コンパイラ エラー C3135
'property': プロパティは 'const' を持つことはできませんまたは 'volatile' 型  
  
 [Const](../../cpp/const-cpp.md)と[揮発性](../../cpp/volatile-cpp.md)プロパティにキーワードは使用できません。

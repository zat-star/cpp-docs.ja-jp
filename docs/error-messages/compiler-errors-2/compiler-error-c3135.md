# コンパイラ エラー C3135
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'property' : プロパティには 'const' または 'volatile' 型は使えません。  
  
 [const](../../cpp/const-cpp.md) キーワードと [volatile](../../cpp/volatile-cpp.md) キーワードは、プロパティには使用できません。

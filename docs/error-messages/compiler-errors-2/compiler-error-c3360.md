# <a name="compiler-error-c3360"></a>コンパイラ エラー C3360
'string': 名前を作成できません  
  
 渡された値、 [uuid](../../windows/uuid-cpp-attributes.md)属性が無効でした。  
  
 次の例では C3360 が生成されます。  
  
```  
// C3360.cpp  
[ uuid("1") ]  
// try this line instead  
// [ uuid("12341234-1234-1234-1234-123412341234") ]  
struct A   // C3360  
{  
};  
  
int main()  
{  
}  
```

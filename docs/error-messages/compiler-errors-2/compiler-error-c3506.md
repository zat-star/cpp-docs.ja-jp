# コンパイラ エラー C3506
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

LIBID 'id' 用に登録された typelib はありません。  
  
 typelib が正しく登録されていません。  regsvr32.exe を使用してタイプ ライブラリを登録してください。

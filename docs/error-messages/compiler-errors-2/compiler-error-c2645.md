# <a name="compiler-error-c2645"></a>コンパイラ エラー C2645
メンバーへのポインターの修飾名がない (見つかった ':: *')  
  
 メンバーへのポインターの宣言は、クラスを指定していません。  
  
 次の例では、C2645 が生成されます。  
  
```  
// C2645.cpp  
class A {};  
int main() {  
   int B::* bp;   // C2645 B not defined  
   int A::* ap;   // OK  
}  
```

# コンパイラ エラー C2874
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

using 宣言によって 'symbol' の多重宣言が発生しました。  
  
 宣言によって同じ項目が 2 回定義されています。  
  
 次の例では警告 C2874 が生成されます。  
  
```  
// C2874.cpp  
namespace Z {  
   int i;  
}  
  
int main() {  
   int i;  
   using Z::i;   // C2874, i already declared  
}  
```

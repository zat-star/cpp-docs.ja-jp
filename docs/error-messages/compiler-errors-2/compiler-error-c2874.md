# <a name="compiler-error-c2874"></a>コンパイラ エラー C2874
using 宣言をによって 'symbol' の複数の宣言  
  
 宣言は、2 回定義する同じ項目です。  
  
 次の例では、C2874 が生成されます。  
  
```  
// C2874.cpp  
namespace Z {  
   int i;  
}  
  
int main() {  
   int i;  
   using Z::i;   // C2874, i already declared  
}  
```

# <a name="compiler-error-c3273"></a>コンパイラ エラー C3273
__finally はアンマネージ コード内の例外ブロックで使用できません。  
  
 次の例では C3273 が生成されます。  
  
```  
// C3273.cpp  
// compile with: /GX  
int main()  
{     
   try  
   {  
   }  
   catch (int)  
   {  
   }  
   __finally   // C3273, remove __finally clause  
   {  
   }  
}  
```

# <a name="compiler-error-c2979"></a>コンパイラ エラー C2979
明示的な特殊化はジェネリックではサポートされていません  
  
 ジェネリック クラスの宣言が正しくありません。  参照してください[ジェネリック](../../windows/generics-cpp-component-extensions.md)詳細についてはします。  
  
## <a name="example"></a>例  
 次の例では C2979 が生成されます。  
  
```  
// C2979.cpp  
// compile with: /clr /c  
generic <>   
ref class Utils {};   // C2979 error  
  
generic <class T>  
ref class Utils2 {};   // OK  
```

# <a name="compiler-error-c3168"></a>コンパイラ エラー C3168
'type': 基になる列挙型の型が正しくありません。  
  
型の指定を基になる、`enum`型が無効でした。 基になる型は、C++ の整数型または対応する CLR 型にする必要があります。  
  
次の例では、C3168 が生成されます。  
  
```  
// C3168.cpp  
// compile with: /clr /c  
ref class G{};  
  
enum class E : G { e };   // C3168  
enum class F { f };   // OK  
```

# <a name="compiler-error-c3156"></a>コンパイラ エラー C3156
'class' : マネージ型または WinRT 型のローカル定義を持つことができません  
  
 関数に、マネージ型または WinRT 型のクラス、構造体、インターフェイスの定義 (宣言) を含めることはできません。  
  
## <a name="example"></a>例  
 次の例では C3156 が生成されます。  
  
```  
// C3156.cpp  
// compile with: /clr /c  
void f() {  
   ref class X {};   // C3156  
   ref class Y;   // C3156  
}  
```

# <a name="compiler-error-c3322"></a>コンパイラ エラー C3322
'property': 属性 'attribute' に対する有効なプロパティではありません。  
  
 無効または認識されないプロパティ (またはパラメーター) が属性に渡されました。 属性のドキュメントを確認してください。

# <a name="compiler-error-c2551"></a>コンパイラ エラーC2551
'void *' 型には明示的なキャストが必要です  
  
 `void` ポインターが暗黙的な変換によって非 void ポインターに割り当てられています。 明示的なキャストを使用する必要があります。

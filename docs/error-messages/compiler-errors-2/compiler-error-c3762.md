# <a name="compiler-error-c3762"></a>コンパイラ エラー C3762
属性 'attribute' を処理できません。  
  
 ユーザー定義の属性から継承する`System.Security.Permissions.SecurityAttribute`セキュリティ属性の定義に使用されています。 このような属性では使用できません、同じアセンブリが定義されています。

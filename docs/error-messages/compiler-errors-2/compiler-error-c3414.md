# <a name="compiler-error-c3414"></a>コンパイラ エラー C3414
'member': インポートされたメンバー関数を定義することはできません  
  
 メンバーは、参照されたアセンブリで定義されているコードで定義されました。  
  
 次の例では、C3414 が生成されます。  
  
```  
// C3414a2.cpp  
// compile with: /clr /LD  
public ref class MyClass {  
public:  
   void Test(){}  
};  
```  
  
 この場合、次のようになります。  
  
```  
// C3414b2.cpp  
// compile with: /clr  
#using <C3414a2.dll>  
  
void MyClass::Test() {    // C3414  
}  
  
System::Object::Object() {    // C3414  
}  
```

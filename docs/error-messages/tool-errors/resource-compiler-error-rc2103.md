# リソース コンパイラ エラー RC2103
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

unexpected end of file in string literal  
  
 文字列が終了せずに、ファイルの終わりが現れました。  文字列の終わりを示す二重引用符 \(**"**\) が抜けている可能性があります。

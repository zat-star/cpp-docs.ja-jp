# <a name="resource-compiler-fatal-error-rc1020"></a>リソース コンパイラの致命的なエラー RC1020
予期しない ' #endif'  
  
 `#endif` 、適合させることがなくディレクティブが登場した`#if`、 **#ifdef**、または**#ifndef**ディレクティブです。  
  
 一致するが確認`#endif`ごとに`#if`、 **#ifdef**、および**#ifndef**ステートメントです。

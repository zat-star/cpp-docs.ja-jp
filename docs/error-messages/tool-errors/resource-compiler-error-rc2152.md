# <a name="resource-compiler-error-rc2152"></a>リソース コンパイラ エラー RC2152
無効なコントロール文字  
  
 **ACCELERATORS** ステートメントの制御文字は無効です。 有効なコントロール文字から成る 1 文字 (のみ)、キャレット (**^**)。

# <a name="expression-evaluator-error-cxx0004"></a>式エバリュエーター エラー CXX0004
構文エラー  
  
 式の構文が正しくありません。  
  
 正しい構文を使用して式を再入力します。  
  
 このエラーは、can0004 と同じものと同じです。

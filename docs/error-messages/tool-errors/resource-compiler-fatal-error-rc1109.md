# リソース コンパイラの致命的なエラー RC1109
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

error creating resource\-name  
  
 指定された .res ファイルを生成できません。  保存先のドライブが書き込み禁止になっていないかどうかを確認してください。  ファイルが生成されたかどうかを確認するには、\/V オプションを使用します。

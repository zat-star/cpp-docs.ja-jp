# リソース コンパイラの警告 RC4214
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Codepage not valid : ignored  
  
 .rc ファイルにコード ページの引数が記述されていますが、指定されたコード ページは無効です。  詳細については、「[IsValidCodePage](http://msdn.microsoft.com/library/windows/desktop/dd318674)」を参照してください。

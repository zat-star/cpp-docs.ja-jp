# <a name="linker-tools-warning-lnk4229"></a>リンカー ツールの警告 LNK4229

 ディレクティブが経由で渡された、[コメント](../../preprocessor/comment-c-cpp.md)プラグマが無効でした。 無視`/directive`です。

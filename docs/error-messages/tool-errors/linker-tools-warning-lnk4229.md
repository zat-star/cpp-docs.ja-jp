# リンカー ツールの警告 LNK4229
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

無効なディレクティブ \/directive が見つかりました。無視します。  
  
 [comment](../../preprocessor/comment-c-cpp.md) プラグマを介して渡されたディレクティブは無効です。  リンカーは `/directive` を無視します。

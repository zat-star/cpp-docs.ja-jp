# <a name="bscmake-warning-bk4503"></a>BSCMAKE の警告 BK4503
.SBR ファイル 'ファイル名' 中に重大でないエラーがありましたが、無視しました。  
  
 .sbr ファイルにエラーが含まれていましたが、ビルドは停止されませんでした。生成された .bsc ファイルが正しくない可能性があります。 この問題を解決するには、再コンパイルして新しい .sbr を作成します。

 既定の言語の指定 (/l) オプションが無効な 16 進数の言語 ID の後に 参照してください[言語および国/地域識別文字列](../../c-runtime-library/locale-names-languages-and-country-region-strings.md)で、*ランタイム ライブラリ リファレンス*有効な言語 Id の一覧についてはします。

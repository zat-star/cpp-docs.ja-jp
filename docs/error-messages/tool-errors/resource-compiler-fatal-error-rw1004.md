# リソース コンパイラの致命的なエラー RW1004
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Unexpected end of file  
  
 このエラーは、テキスト ファイルの最終行にライン フィード文字および復帰文字がないことが原因で発生する場合があります。

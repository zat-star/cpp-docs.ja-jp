# <a name="resource-compiler-error-rc2162"></a>リソース コンパイラ エラー RC2162
文字列化演算子 (#) の後のトークンは、仮引数名でなければいけません。  
  
 文字列化演算子トークン (**#**) 仮パラメーター名ではありません。

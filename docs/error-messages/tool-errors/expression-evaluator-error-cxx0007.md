# <a name="expression-evaluator-error-cxx0007"></a>式エバリュエーター エラー CXX0007
右かっこがありません。  
  
 式では、対になっていないかっこが見つかりました。 バランスの取れたかっこ付きの式を再入力します。  
  
 このエラーは、can0007 と同じものと同じです。

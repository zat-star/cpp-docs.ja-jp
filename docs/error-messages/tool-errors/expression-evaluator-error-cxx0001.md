# 式エバリュエーター エラー CXX0001
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

ユーザー関数の実行中に例外がおきました  
  
 プログラム コード実行中に一般保護違反が発生しました。  このエラーは CAN0001 と同じものです。

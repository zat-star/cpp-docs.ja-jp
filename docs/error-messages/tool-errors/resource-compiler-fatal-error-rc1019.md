# <a name="resource-compiler-fatal-error-rc1019"></a>リソース コンパイラの致命的なエラー RC1019
予期しない ' #else '  
  
 `#else`ディレクティブが内に存在しません、 `#if`、 **#ifdef**、または**#ifndef**を構築します。  
  
 あることを確認してください、 `#if`、 **#ifdef**、または**#ifndef**ステートメントの次のステートメントの前に有効にします。

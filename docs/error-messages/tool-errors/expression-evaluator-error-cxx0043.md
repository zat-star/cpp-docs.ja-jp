# <a name="expression-evaluator-error-cxx0043"></a>式エバリュエーター エラー CXX0043
このポインターはメンバー関数の外で使用  
  
 **この**ポインターは、非静的メンバー関数に対してのみ使用できます。  
  
 このエラーは、can0043 と同じものと同じです。

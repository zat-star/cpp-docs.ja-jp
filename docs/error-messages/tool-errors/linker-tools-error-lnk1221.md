# <a name="linker-tools-error-lnk1221"></a>リンカ ツール エラー LNK1221
サブシステムは推論することはできませんし、定義する必要があります。  
  
 リンカーでは、対象となるサブシステムを推論するのに十分な情報はありません。  
  
 このエラーを解決するには使用[/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)です。

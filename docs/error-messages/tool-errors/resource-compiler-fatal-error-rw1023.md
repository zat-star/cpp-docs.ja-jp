# <a name="resource-compiler-fatal-error-rw1023"></a>リソース コンパイラの致命的なエラー RW1023
ドライブがいっぱいにファイルを書き込み中に I/O エラー  
  
 空き領域を作成する実行可能ファイルのサイズの少なくとも 2 倍に等しくなければなりません。

# <a name="expression-evaluator-error-cxx0028"></a>式エバリュエーター エラー CXX0028
構造体/共用体の要素ではありません  
  
 'メンバー-> pStruct' または '体' フォームの式が指定されましたが、\<メンバー > 構造体の要素ではありません。  
  
 式はかっこで囲まれたを正しくされていない可能性があります。  
  
 このエラーは、can0028 と同じものと同じです。

# リソース コンパイラの致命的なエラー RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

コンパイラの制限 : マクロの入れ子が深すぎます  
  
 マクロの入れ子がリソース コンパイラの制限を超えています。  ソース ファイルを修正し、マクロの入れ子の深さを減らしてください。

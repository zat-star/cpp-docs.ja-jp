# <a name="resource-compiler-error-rc2167"></a>リソース コンパイラ エラー RC2167
認識できない VERSIONINFO フィールドです。BEGIN またはコンマが必要  
  
 認識できないフィールドが見つかりました、**固定**の一部、 **VERSIONINFO**構造体宣言します。 A **VERSIONINFO**フィールドはコンマで区切られた DWORD でなければなりません。

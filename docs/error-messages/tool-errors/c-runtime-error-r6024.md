# C ランタイム エラー R6024
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

not enough space for \_onexit\/atexit table  
  
 `_onexit` 関数と `atexit` 関数に使用できるメモリがありません。  このエラーはメモリが不足しているときに発生します。

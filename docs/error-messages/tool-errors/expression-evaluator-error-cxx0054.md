# <a name="expression-evaluator-error-cxx0054"></a>式エバリュエーター エラー CXX0054
ブレークポイント式のレジスタは無効です  
  
 レジスタは、ブレークポイント式では使用できません。  
  
 このエラーは、can0054 と同じものと同じです。

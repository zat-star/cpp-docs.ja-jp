# BSCMAKE エラー BK1510
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

.SBR ファイル 'filename' は壊れています。  
  
 指定した .sbr ファイルは正しい形式ではありません。  再コンパイルして、.sbr ファイルを作成し直してください。

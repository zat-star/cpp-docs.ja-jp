# リソース コンパイラ エラー RC2124
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

empty menus not allowed  
  
 **MENU** ステートメントでメニュー項目を 1 つも定義しないうちに **END** キーワードが現れました。  空のメニューは使用できません。  **MENU** ステートメントの引用符の対応関係が正しいかどうかを確認してください。

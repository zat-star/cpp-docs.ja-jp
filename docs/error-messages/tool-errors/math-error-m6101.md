# <a name="math-error-m6101"></a>数値演算エラー M6101
無効な  
  
 操作が無効です。  
  
 このエラーは、オペランドが NaN (非数) または無限大の場合に発生することができます。  
  
 プログラムは、終了コード 129 で終了します。

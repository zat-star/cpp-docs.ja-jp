# リンカー ツールの警告 LNK4194
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

\/DELAYLOAD:dll name は無視されます。  
  
 リンカーは指定された DLL を[遅延読み込み](../../build/reference/delayload-delay-load-import.md)できません。

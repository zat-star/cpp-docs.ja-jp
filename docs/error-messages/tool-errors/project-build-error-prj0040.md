# <a name="project-build-error-prj0040"></a>プロジェクト ビルド エラー PRJ0040
ビルドで内部エラー。 続行できません。 プロジェクトを再度読み込んで、再試行してください。  
  
 プロジェクト システムは、プロジェクトを処理できませんでした。  
  
 プロジェクトの再読み込みを再試行してください。

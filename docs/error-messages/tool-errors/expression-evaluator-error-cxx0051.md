# 式エバリュエーター エラー CXX0051
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

1 つ以上のオーバーロードされたシンボルがブレークポイントで指定されています  
  
 シンボルが 2 回以上オーバーロードされているため、ブレークポイントとして使用するシンボルが決定できません。  
  
 このエラーは CAN0051 と同じものです。

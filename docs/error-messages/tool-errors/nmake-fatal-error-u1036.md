# NMAKE の致命的なエラー U1036
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

構文エラー : '\=' の左に名前が多すぎます。  
  
 マクロ定義の左側に置くことができる文字列は 1 つだけです。

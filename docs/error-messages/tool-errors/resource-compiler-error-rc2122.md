# リソース コンパイラ エラー RC2122
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

unknown menu subtype  
  
 **MENU** ステートメントの *item\-definition* フィールドに指定できるのは、**MENUITEM** ステートメントと **POPUP** ステートメントだけです。

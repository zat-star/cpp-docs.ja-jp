# <a name="resource-compiler-error-rc2107"></a>リソース コンパイラ エラー RC2107
**数値コマンドの値が必要**  
  
 リソース コンパイラは型の数値を予期していた*idvalue*フィールドに、**アクセラレータ**ステートメントです。 使用しているかどうかを確認、`#define`定数値を指定する定数のスペルが正しいこととします。

# 数値演算エラー M6107
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

unemulated  
  
 無効なコプロセッサ命令、またはエミュレーターがサポートしないコプロセッサ命令を実行しようとしました。  
  
 プログラムは終了コード 135 で終了します。

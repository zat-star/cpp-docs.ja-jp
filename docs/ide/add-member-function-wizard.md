# <a name="add-member-function-wizard"></a>メンバー関数の追加ウィザード
このウィザードでは、ヘッダー ファイルと、選択したクラスの実装ファイルにスタブ メンバー関数の実装にメンバー関数の宣言を追加します。  
  
 このメンバー関数は、ウィザードを使用して、追加した後は、開発環境でコードを編集できます。  
  
 **戻り値の型**  
 追加するメンバー関数の戻り値の型を設定します。 戻り値の型を提供できます。 または使用可能な種類の一覧から選択できます。 型については、次を参照してください。[基本的な型](../cpp/fundamental-types-cpp.md)です。

|-|-|-|  
|`char`|`int`|`unsigned int`|  
|**double**|**long**|`unsigned long`|  
|**float**|**short**|`void`|  
|`HRESULT`|`unsigned char`||  
  
 **関数名**  
 追加するメンバー関数の名前を設定します。  
  
 **パラメーターの型**  
 このメンバー関数にパラメーターがある場合は、メンバー関数に追加するパラメーターの型を設定します。 独自のパラメーターの型を提供できます。 または使用可能な種類の一覧から選択できます。  
  
||||

|`char`|`int`|`unsigned char`|

 **パラメーター名**  
 このメンバー関数にパラメーターがある場合は、メンバー関数に追加するパラメーターの名前を設定します。  
  
 **パラメーター リスト**  
 このメンバー関数に追加したパラメーターの一覧を表示します。 一覧に、パラメーターを追加する、型を指定し、名前、**パラメーターの型**と**パラメーター名**ボックスとクリック**追加**です。 削除するパラメーター一覧からパラメーターを選択し、 をクリックして**削除**です。  
  
 **アクセス**  
 メンバー関数へのアクセスを設定します。 アクセス修飾子は、他のクラス メンバー関数へのアクセスを指定するキーワードです。 参照してください[メンバー アクセス コントロール](../cpp/member-access-control-cpp.md)詳細については、アクセスを指定します。 メンバー関数のアクセス レベルに設定**パブリック**既定です。  
  
-   [public](../cpp/public-cpp.md)  
  
-   [protected](../cpp/protected-cpp.md)  
  
-   [private](../cpp/private-cpp.md)  
  
 新しいメンバー関数は、静的または仮想、かどうかと、インラインであるかどうかを確認または純粋なです。 メンバー関数を純粋関数を設定した場合、 `Virtual`  チェック ボックスをオン、**インライン** チェック ボックスは使用できなくなります。 既定では、非静的、非仮想メンバー関数です。  
  
|オプション|説明|  
|------------|-----------------|  
|[Static](../cpp/storage-classes-cpp.md)|関数がグローバルのように動作し、ないクラスのインスタンス化した場合でも、クラスの外部で呼び出すことができることを指定します。 このメンバー関数には、非静的メンバーへのアクセスがありません。 として指定されたメンバー関数`Static`仮想にすることはできません。|  
|[仮想](../cpp/virtual-cpp.md)|そのメンバー関数の呼び出しに使用する式に関係なく、オブジェクトの適切なメンバー関数が呼び出されることを確認します。 として指定されたメンバー関数`Virtual`静的にすることはできません。|  
|**純粋です**|宣言されている; の仮想メンバー関数の実装が指定されていないことを示しますしたがって、**純粋**仮想メンバー関数にのみ指定できます。 少なくとも 1 つの純粋仮想メンバー関数を含むクラスは抽象クラスと見なされます。 抽象クラスから派生したクラスは純粋仮想メンバー関数を実装する必要がありますか、すぎる、抽象クラスです。|  
|[インライン](../cpp/inline-functions-cpp.md)|メンバー関数が呼び出される各位置に、メンバー関数本体のコピーを挿入するようにコンパイラに指示します。 として指定されたメンバー関数**インライン**純粋にすることはできません。|  
  
 **.cpp ファイル**  
 スタブ メンバー関数の実装の書き込み先ファイルの場所を設定します。 既定では、このメンバー関数を追加するクラスの .cpp ファイルに書き込まれます。 ファイル名を変更する、省略記号ボタンをクリックします。 メンバー関数の実装は、選択したファイルの内容に追加されます。  
  
 **コメント**  
 このメンバー関数用のヘッダー ファイル内のコメントを提供します。  
  
 **関数シグネチャ**  
 このメンバー関数が表示されます をクリックすると、コードに表示される**完了**です。 このボックスにテキストを編集することはできません。 このメンバー関数を変更するには、ウィザードで、該当するボックスを変更します。
